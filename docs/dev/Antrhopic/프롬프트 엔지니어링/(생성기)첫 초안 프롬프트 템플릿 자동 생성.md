# 첫 초안 프롬프트 템플릿 자동 생성

<Note>
  우리의 프롬프트 생성기는 확장된 사고 능력을 가진 모델을 포함한 모든 Claude 모델과 호환됩니다. 확장된 사고 모델에 특화된 프롬프팅 팁은 [여기](/ko/docs/build-with-claude/extended-thinking)를 참조하세요.
</Note>

때로는 AI 모델을 사용할 때 가장 어려운 부분이 효과적으로 프롬프트하는 방법을 찾는 것입니다. 이를 돕기 위해, 우리는 Claude가 귀하의 특정 작업에 맞춤화된 고품질 프롬프트 템플릿을 생성하도록 안내하는 프롬프트 생성 도구를 만들었습니다. 이러한 템플릿들은 우리의 많은 프롬프트 엔지니어링 모범 사례를 따릅니다.

프롬프트 생성기는 특히 "백지 상태의 문제"를 해결하고 추가 테스트와 반복을 위한 시작점을 제공하는 도구로서 유용합니다.

<Tip>지금 바로 [Console](https://console.anthropic.com/dashboard)에서 프롬프트 생성기를 사용해보세요.</Tip>

기본 프롬프트와 아키텍처를 분석하는 데 관심이 있다면, 우리의 [프롬프트 생성기 Google Colab 노트북](https://anthropic.com/metaprompt-notebook/)을 확인해보세요. 거기서 Claude가 귀하를 대신하여 프롬프트를 구성하도록 하는 코드를 쉽게 실행할 수 있습니다.

<Note>Colab 노트북을 실행하려면 [API 키](https://console.anthropic.com/settings/keys)가 필요합니다.</Note>
